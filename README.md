#Triper React


## 프로젝트
create-react-app + typescript 국내여행지 찾기 토이 프로젝트

#### 추가 패키지
- recoil ([https://recoiljs.org](https://recoiljs.org)) - React 상태관리 패키지
- styled-components ([https://styled-components.com](https://styled-components.com)) - 컴포넌트 스타일링 패키지
- classnames ([https://www.npmjs.com/package/classnames](https://www.npmjs.com/package/classnames)) - 클래스의 조건처리를 편하게 하기 위한 패키지
- craco ([https://github.com/gsoft-inc/craco](https://github.com/gsoft-inc/craco)) - React를 eject 하지 않고 설정을 하기 위한 패키지
- qs ([https://www.npmjs.com/package/qs](https://www.npmjs.com/package/qs)) - 쿼리스트링을 편하게 사용하기 위한 패키지
- node-sass@4.14 - 버전 이슈로 인해 4.14로 설치

#### 시작
```bash
$ npm install       // 프로젝트 설치
```
```bash
$ npm run start     // 프로젝트 시작 (localhost:3000)
```
